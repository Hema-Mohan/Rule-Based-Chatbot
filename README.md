# 🤖 Rule-Based Chatbot (Task 8)

## 📌 Objective
This project is a simple rule-based chatbot built using Python.  
It uses basic control flow statements like if-elif-else and loops to simulate conversation.

## 🛠 Tools Used
- Python 3

## 💡 Key Concepts
- Control Flow (if-elif-else)
- Loops (while loop)
- User Input Handling
- String methods (lower())

## 🚀 Features
- Responds to greetings (hi, hello)
- Answers basic questions
- Provides help command
- Exits when user types "exit"
- Handles unknown inputs
